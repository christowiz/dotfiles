# OS X Dotfiles Installation

Shell script to install system dotfiles


## Easy one-liner

```
curl -sSL "https://raw.githubusercontent.com/christowiz/dotfiles/master/bootstrap.sh" | sh
```


## Download repo/file and run locally

```
$ sh bootstrap.sh
```
